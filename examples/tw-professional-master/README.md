# Taiwan Professional Master's Mini Sample

This folder is a public-safe mini sample for a Taiwan professional master's student who starts from a vague workplace topic.

It demonstrates the workflow shape only. It does not contain real student writing, advisor feedback, private data, or verified literature claims.

## Sample Topic

Raw topic: `荳ｭ蟆丈ｼ∵･ｭ ESG 蟆主Ş蝗ｰ髮｣`

The point is not ESG itself. The point is to show how any vague professional-practice topic can be turned into:

1. a narrowed topic brief,
2. a literature matrix,
3. a gap radar,
4. advisor-facing questions,
5. chapter-level writing tasks.

## Files

- `topic-brief.md`: first-pass topic clarification.
- `literature-matrix.csv`: placeholder matrix rows with `unverified` status.
- `gap-radar.md`: candidate gaps to test with sources and advisor feedback.
- `advisor-feedback-log.md`: a simple way to track what the advisor accepted or rejected.
- `chapter-task-map.md`: how to turn the research direction into chapter tasks with paragraph IDs.
