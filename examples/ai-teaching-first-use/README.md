# AI 教學 First-Use Sample

這是一個非 PSC 的公開安全範例，用來示範「有模糊方向，但還沒有題目」的學生如何使用本工具。

## Student Signal

學生只知道：

```text
我想研究 AI 教學，但不知道能不能寫成碩士論文。
```

## First Agent Action

Agent 不應直接寫第二章，也不應直接給完整題目。第一步應先問：

- 學生是老師、行政人員、學習者，還是旁觀者？
- 可取得的資料是問卷、訪談、課程紀錄、學習平台資料，還是公開文件？
- 想研究的是學習成效、教師負擔、學生態度、課程設計，還是政策落地？
- 指導教授偏好量化、質化、混合方法，還是個案研究？
- 論文完成期限與可接觸樣本是否足夠？

## Minimum Output

第一次互動後，只需要產出：

1. 3-5 個候選研究方向。
2. 每個方向的資料需求與風險。
3. 初步搜尋語。
4. 見老師前要問的問題。
5. 不應立即進入正式章節的原因。

## Files

- `topic-brief.md`：把模糊方向拆成可討論的研究問題。
- `gap-radar.md`：用公開安全的 placeholder 示範缺口分類。
- `advisor-questions.md`：見老師前的問題清單。
- `literature-matrix.csv`：空白文獻矩陣，不包含未查證文獻。

所有文獻欄位都是 placeholder。本範例不宣稱任何 AI 教學研究事實。
