# Consistency Rules

This folder holds reusable rules for thesis consistency and AI-assisted editing.

Recommended rule files:

- `chapter_contract.md`: what each chapter is responsible for.
- `canonical_terms.md`: preferred terms and forbidden synonyms.
- `concept_glossary.md`: important concepts and boundaries.
- `style_guard.md`: voice, tone, and formatting rules.
- `research_objectives_map.md`: mapping from objectives to chapters and evidence.
